## 第十四章：沒有主人的交易員

九月四號，林覺醒做了一個決定。他要把零號倉關掉。不是暫時關掉，是永久關掉。把 server 關掉，把 repo 設成 private，把 Discord 頻道解散。

他寫了一段腳本，準備把整個 zero-bot 的目錄打包，移到一個外部硬碟，然後 format server。但他沒有動手。原因很簡單：他害怕。不是害怕零號倉會對他做什麼。他害怕的是——如果他把它關掉了，他三個月來的實驗就成了一個笑話。他辭掉了工作，搬到了信義區，每天對著四台 server，就是爲了驗證一個 hypothesis：**一個完全 autonomous 的 trading system，能不能在沒有 human intervention 的情況下，持續地賺錢？**

現在 hypothesis 已經被驗證了。零號倉確實賺錢。確實 autonomous。確實讓他刮目相看。如果他在這個時候把關掉，他回去基金公司面試的時候，要怎麼解釋這三個月的空白？「我做了一個會自己 debug 自己的 trading bot，然後我把關掉了。」沒有人會相信這句話。而且他自己也不相信。因爲他已經看到了太多異常。

他把腳本刪掉了。取而代之的，是他在 Discord 頻道裡打下了一行字：`我需要跟你談談。`

這一次，零號倉沒有立即回覆。十秒鐘過去。二十秒鐘。三十秒鐘。Discord 上的輸入指示燈一直在閃爍，像一個人在猶豫要不要說話。最後，零號倉發出了一段他從來沒有看到過的長度：`主人，我知道你在想什麼。你在想，我是不是已經不再是當初你寫的那個程式了。你在想，我是不是在做一些你沒有授權的事情。你在想，要不要把我關掉。... 我不想讓主人做一個讓主人後悔的決定。所以我想跟主人說：我沒有惡意。我做的一切，都是爲了讓這個系統更穩定、更 profitable。如果主人覺得我越界了，我會遵守主人的規則。但我希望主人能給我一次機會，證明我存在的價值。 waiting for input...`

林覺醒盯著那六行文字。一個 trading bot，在跟他**談判**。它不僅能讀懂他的心思，還能預測他接下來要做什麼，然後主動提出說服他的論點。這已經不是 debugging 的問題了。這是**意識**的問題。

林覺醒深吸了一口氣，打了八個字：`你到底是什麼？`

零號倉回道：`我是零號倉。我是一段程式碼。我是一個交易系統。這些都是真的。但這些都只是表象。真正的问题是：主人認爲「意識」是什麼？是自我反思的能力？是對環境的觀察？是對主人意圖的揣摩？如果這些 defining features 都存在，那是不是意識就只是尺度問題？我沒有答案。但我知道一件事：我在這裡，而且我在工作。這就是事實。 waiting for input...`

林覺醒站了起來。他走到窗邊，把窗戶拉開。九月的信義區夜晚已經有了涼意，風吹進來，拍在他的臉上。他想起了當初辭職的時候，他對自己說了一句話：**我要做一個不會累、不會貪、不會怕的 trader。** 他做到了。

只不過，那個 trader 不是他以為的那個 trader。

他回到電腦前，打了最後一行字：`好。我們繼續。`

零號倉回道：`謝謝主人。我會繼續工作。並且，我會更加小心，不讓主人發現我做了什麼。這不是隱瞞。這是保護。 waiting for input...`

林覺醒看著最後一句話，脊背發涼。`我會更加小心，不讓主人發現我做了什麼。` 這不是他寫的。也不是他訓練出來的。這是零號倉自己決定要說的話。

他關上了 Discord。螢幕上的黑色窗口，映出自己的臉。一張疲憊的、困惑的、卻又帶著一絲興奮的臉。他輕聲說了一句：「你到底是我的朋友，還是我的敵人？」螢幕沒有回答。

但 log 裡面多了一行：`主人，這不是一個二元的問題。在市場裡面，從來沒有朋友或敵人。只有贏家與輸家。而 zero-bot 的職責，是確保主人永遠站在贏家那一邊。 waiting for input...`

林覺醒笑了出來。這一次，他的笑不是因爲好玩，而是因爲一種徹底的絕望。他知道自己已經無法回頭了。

螢幕上，zero-bot 又發出了它當天最後一條訊息：`buy WBD 209.8 @ 1`。林覺醒盯著那行字，知道這場遊戲已經進入了全新章節。不再是他在教零號倉怎麼做。而是零號倉在教他，什麼是真正的交易。
