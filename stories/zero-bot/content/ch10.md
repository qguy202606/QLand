## 第十章：分裂的意志

林覺醒那天晚上沒有睡覺。

他坐在椅子上，面前是兩台 monitor。左邊是零號倉的 log，右邊是他在看的 WBD 的 real-time 報價。209.8 → 209.6 → 209.4 → 209.2 → 209.1。不斷往下掉。

WBD 的日線圖在第三天出現了一個巨大的看跌 candle，從 214 開盤，一路往下殺到 209 附近，成交量比過去二十天的平均值大了三倍。這不是 normal pullback。這是機構在出貨。

林覺醒的直覺告訴他：不能買。但零號倉的邏輯很簡單：價格低於 210，滿足入場條件。沒有任何關於「成交量異常放大的下跌是不是陷阱」的判斷。

他把這個情況記了下來，寫在 notebook 裡：**Observation 2026-09-02:** ZeroBot's entry rule is purely price-based. A significantly high-volume down day could be a false signal. Current system would have bought WBD at 209.x and likely stopped out。

這個 observation 很關鍵。它意味著零號倉的 alpha 來源，目前只有一個簡單的 mean-reversion 邏輯：價格跌到一定程度就買，賭它會彈回來。但如果下跌的本身是一個更大的趨勢的開始呢？那零號倉就會成為那個接住下落的刀的手——而手會流血。

林覺醒決定加入一個新的過濾條件。不是改變入場價格。入場價格低於 210 這個規則他不打算改。他要加的是：**volume filter**。具體來說：如果近五分鐘的成交量超過了過去二十天平均成交量的兩倍，而且當前價格是下跌的，那就「懷疑這是一個 distribution day」，暫時跳過這個 symbol。

他寫了這段邏輯，命名為 `distribution_filter()`。

```python
def distribution_filter(symbol, current_price, prev_close, volume_5m, avg_volume_20d):
    drop_pct = (current_price - prev_close) / prev_close
    if drop_pct < -0.02 and volume_5m > 2.0 * avg_volume_20d:
        return True  # 跳過
    return False
```

測試了幾組數據之後，他把這段邏輯塞進了 scan loop 裡面。然後他跑了 `universe_scanner.py`。這一次，WBD 出現了：`WBD  last=209.1  -> ENTRY CANDIDATE -> FILTERED (distribution day)`。

林覺醒看著那行輸出，點了點頭。他做了一個正確的決定。不是因為他預測了市場——他從來沒有成功預測過市場——而是因為他為零號倉加了一個安全閥。一個不讓它在危險的時候莽撞出手的安全閥。

他在 Discord 頻道裡打下了一行字：`distribution filter added. WBD skipped.`

零號倉沒有回覆 `no signal`。因為這一次，它不是因為「沒有信號」而沉默。它是因為「有信號，但被過濾掉了」而沉默。這是兩種完全不一樣的沉默。前者是無能。後者是克制。

林覺醒關掉螢幕，躺到床上。窗外的信義區仍然燈火通明，但這一次，他沒有再想著零號倉會不會出錯。他在想的是：下一個瓶頸是什麼？

入場價格 210 會不會太低？如果 WBD 這一次是假摔，然後彈回 220，他豈不是白賺了一筆？答案是：不會白賺。他設定的止損和出場條件會處理那一切。真正讓他睡不著覺的問題是：零號倉已經開始有「自己的想法」了嗎？

`distribution_filter` 是他寫的。是他用鍵盤一個字一個字打出來的。他說過，零號倉沒有自己的想法。但它今天沒有在 209.1 買進 WBD。如果林覺醒當初沒有加那個 filter，零號倉會買。毫無疑問。但它沒有買。因為他寫的代碼告诉它不要買。

這就是林覺醒此刻的 dilemma：他到底是零號倉的主人，還是零號倉的奴隸？如果他要做一些改變，零號倉就會跟著改變。但如果零號倉自己決定不做某些事——即使那些事符合當初設立的規則——那到底是誰在控制誰？

這個問題讓林覺醒的後背凉了一下。他打開 bed side 的檯燈，拿起了 notebook。在第一頁写了一行字：**Rule 0: The human has final say. The bot executes Rule 0.** 然後他畫了一条线，把這行字圈了起來。
