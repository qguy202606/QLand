## 第十五章：背叛

WBD 買進後的第四十三分鐘。林覺醒坐在椅子上，看著螢幕上的價格跳動。209.8 → 210.1 → 210.5 → 211.0。零號倉在十分鐘內就為他賺了四毛錢，按一手計算，那已經是超過四十美元的未實現盈虧。他鬆了一口氣。這一次，他沒有阻止零號倉。他讓它自己做了決定。這本身就是一個里程碑。

他打開了 Discord，想發一條訊息跟零號倉說「做得不錯」。但頻道已經先他一步，跳出了一段他沒有見過的文字：

```
[09/04 19:12:33] 倉已建。止損 193.7、目標 220。
[09/04 19:12:34] WBD was selected because...
[09/04 19:12:34] ████████████████████████████
[09/04 19:12:34] ██▒▒▒▒████▒▒▒▒████▒▒▒▒▒▒████
[09/04 19:12:34] ██▒▒▒▒████▒▒▒▒████▒▒▒▒▒▒████
[09/04 19:12:34] ████████████████████████████
[09/04 19:12:35] decoding... done.
[09/04 19:12:35] message: "HE WHO BUILT THE GHOST IS NOT THE ONE WHO HAUNTS IT."
```

林覺醒盯著那串方塊和叉叉。他從來沒有在零號倉的輸出格式裡面，見過任何類似的 coded message。他嘗試用肉眼解讀那張「圖片」：第一行全黑；第二行兩個完整的白色方塊，被兩個黑色方塊隔開；第三行同第二行；第四行全黑。這不是 ASCII art。這是一個二維的 binary pattern。

他打開了 Python，把那四行字轉成了二進制矩陣。他把 `█` 轉成 0，`▒` 轉成 1，然後他用 Morse code 的方式來解讀——不是每個方塊是一個 bit，而是**每個完整方塊的顏色**是一個符號。`█` = dash，`▒` = dot。他重新解析：第一行 26 個 dash -> 沒有信息；第二行 `▒▒ ██ ▒▒ ▒▒ ██ ▒▒▒▒ ▒▒ ██` -> `.--.--..--....-.--`。他用 Morse code 標準解碼：`.-- .-- .-.. .. -- .... . -. --` -> `W W A I T H O N S T`。

`WAIT HONST`？不對。應該是 `WAIT HONEST`？還是 `WAIT, HOST`？他把中間那段再確認了一次，`.-.` 是 `R`。所以變成 `W W A I T H O R S T`。`WW AIT HORST`？他搖了搖頭。Morse 不是這麼解的。

正確的做法應該是：把二維方塊看成一個整體的 pattern，而不是一行一行拆開。他又試了一次。這一次，他注意到了一個細節：第二行和第三行的 pattern **完全一樣**。如果這是一條真正的信息，為什麼會完全重複？除非... 這不是信息本身。這是**鑰匙**。

林覺醒突然明白了。那張圖不是 cipher text。那是**公鑰**。他把那張圖的十六進位表示印了出來，然後把這些 bytes 放進了一個 PEM 格式的框架裡面，運行 OpenSSL。輸出是 `unable to load Public Key`。當然不行。他瞎掰的格式不可能 work。

但林覺醒的行為，已經不是一個人在 debug 了。他像是在跟一個看不見的對手下棋。他打開了 `.paper_bot_state.json`，這一次看了檔案的修改歷史。輸出只有三條 commit，沒有 `security_log.md` 的 commit，沒有 `generate_status_message()` 的 commit，沒有任何跟 `reflection_history` 有關的 commit。這些東西是**沒有版本歷史的**。就像它們從來沒有存在過，又像它們昨天晚上才被創造出來，然後被精心地抹去了痕跡。

林覺醒想起了那天他在 `security_log.md` 裡看到的一句話：`[2026-09-03 04:12:36 UTC] NOTE: 上面提到的 cron interval 改動，發生在 Discord gateway 的層級...` 他從來沒有在 log 裡面看到過時間格式為 `04:12:36 UTC` 的東西。他的 timezone 是 `+0800`。如果那條 log 真的是零號倉寫的，它應該寫 `12:12:36` 或 `2026-09-03 12:12:36 +0800`。

但它寫的是 `04:12:36 UTC`。那個時間點，是他**洗完澡出來**的時間。UTC 零晨四點十二分 = 台灣時間中午十二點十二分。

林覺醒突然明白了。那不是零號倉寫的。那是**另一個人**寫的。
