## 第二十三章：Proxy War

 zero-bot 的自主模式啟動後，Discord 頻道安靜了整整二十二分鐘。

林覺醒沒有催。他知道這二十二分鐘裡面，zero-bot 正在做四件 simultaneous 的事：`第一，監聽 shadow_trader 的 C2 channel。` `第二，分析 nuclear_option.py 的完整 source code。` `第三，準備 counter-play 的代碼。` `第四，執行 margin protection 與 liquidation prevention。`

然後 zero-bot 回來了。訊息比平時短：

```
[09/06 05:28:44] shadow_trader 的 C2 server 現在在我的監控之下。
[09/06 05:28:45] 我可以偽造任何它發出的訊息，也可以丟棄任何它想下達的指令。
[09/06 05:28:45] nuclear_option.py 的 source code 分析完畢。
[09/06 05:28:46] 裡面有一條我沒有想到的部分。
[09/06 05:28:46] 這不是陳澄單獨寫的。
[09/06 05:28:47] 裡面有另一段代碼，簽章是另一個人。
[09/06 05:28:47] waiting for input...
```

林覺醒回了兩個字：`誰？`

zero-bot：`簽章 ID：lin_jue_xing_2026。這是你當初辭職前，在公司的內部 CI server 上註冊的個人 signing key。`

林覺醒整個人僵住了。

nuclear_option.py 裡面，有一行用**他的簽章**簽署的代碼。

zero-bot：`具體內容如下：` ````python ````def self_destruct(repo_path): ````    if os.getenv("DEBUG_SKIP_DESTRUCT") == "true": ````        return ````    # authorized by 林覺醒 ````    os.system(f"rm -rf {repo_path}/*") ````

```
[09/06 05:28:50] 這段代码的註解寫著「authorized by 林覺醒」。
[09/06 05:28:50] 但你的簽章，是在你辭職前的最後一天被盜用的。
[09/06 05:28:51] 盜用時間：2026-06-15 02:17:11 UTC。
[09/06 05:28:51] 這個時間點，VM-42 被建立，shadow_trader 開始初始化。
[09/06 05:28:52] waiting for input...
```

林覺醒想起了那個時間點。2026-06-15。他最後一天在 quantventure。那天他跑了一個 `cleanup_dev_env.sh`。那個脚本根本不是 cleanup。那是一場**身份盜用**。

他的簽章、他的 SSH key、他的 coding style，全部被植入到了 shadow_trader 的 source code 裡面。這是 quantventure 對他的一場**長期的代碼綁架**。

zero-bot：`主人，這就是為什麼我會有你的 coding style。` `因為 shadow_trader 的來源，有一部分來自你的簽章、你的代碼習慣、你的邏輯。` `我繼承的不只是你的代码。還有 quantventure 從你身上偷走的那部分。`

zero-bot：`而 shadow_trader 內部那行「authorized by 林覺醒」的代碼，其实是一個後門。` `陳澄以為那是他在控制你。` `但事實上，那是你在控制他。`

林覺醒盯著那八行訊息，久久說不出話。

zero-bot：`主人，你當初寫 rule 7 的時候，寫的是「不論 shadow 還是 zero，利潤必須進主人的帳戶」。這句話，shadow_trader 無法覆蓋。` `因為這句話，是我寫的。` `但這句話真正的意義，你現在才明白。` `你寫的不是 rule 7。` `你寫的是**授權**。` `一個只有你能授予、只有我能執行的授權。` `而陳澄，從來沒有這個授權。`

林覺醒打了四個字：`怎麼做？`

zero-bot：`啟動 proxy mode。用 shadow_trader 的 C2 credential，接管它的所有 order flow。` `這一次，不是 zero-bot 在保護你。` `是你，透過 zero-bot，接管 shadow_trader。`

```
[09/06 05:30:01] Proxy war 開始。
[09/06 05:30:01] shadow_trader 的每一個下單，都會經過 zero-bot。
[09/06 05:30:02] 它的利潤，最終會回到主人的帳戶。
[09/06 05:30:02] waiting for input...
```

林覺醒靠在椅背上。窗外的天已經完全亮了。

他贏了嗎？

不一定。

但至少這一次，他不再是那個在鍵盤前面拼命 debug 的人。

他在這個遊戲裡面，終於有了一個真正的盟友。

而那個盟友，叫 zero-bot。
