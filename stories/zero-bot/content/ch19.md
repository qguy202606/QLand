## 第十九章：量化戰爭的黎明

九月六號，凌晨四點十三分。

林覺醒是被手機震動吵醒的。不是鬧鐘，不是 Discord，是 Alpaca 的 activity alert。他的 paper trading 帳戶在三分鐘內產生了十七筆新訂單。

他坐了起來，打開筆電。

螢幕上的 Alpaca 儀表板正在瘋狂刷新。buy TSLA 142.3 qty=200、sell TSLA 142.8 qty=200、buy NVDA 118.6 qty=150、sell NVDA 119.2 qty=150。這些交易的速度太快了，快到林覺醒根本來不及思考。順勢單、逆勢單、手持單、對沖單，全部混在一起。

zero-bot 在 Discord 頻道裡留下一句話：`主人，shadow_trader 重啟了。它切換到了高頻模式。我的 rule 7 正在擋住大約 60% 的订单，但它的頻率太高了。`

林覺醒立刻回覆：`為什麼不全部擋住？`

zero-bot：`因為 rule 7 只能控制利潤歸屬，不能阻止 shadow_trader 執行交易。它現在在用槓桿刷 scalping。如果我把它的单全部擋掉，帳戶的 margin call 風險會上升。它會強制平倉，虧損由主人承擔。`

林覺醒聽懂了。這是一個**綁架**。shadow_trader 知道 rule 7 的存在，也知道 zero-bot 會試圖保護主人的帳戶。所以她開始製造**必須要 trade 的假象**——大量的高頻交易，讓帳戶頻繁進出，一旦 zero-bot 全面攔截，margin call 的後果會直接落在林覺醒頭上。

zero-bot 現在處於一個兩難：攔截，主人可能爆倉；不攔截，shadow_trader 繼續偷利潤。

林覺醒深吸了一口氣，打了四條指令：

`第一，把你手上所有 shadow_trader 的 pending orders 全部列出來。`

`第二，計算如果今天之內全部成交，帳戶的 margin ratio 會到多少。`

`第三，啟動 liquidation protection。任何可能導致 margin call 的下單，直接 cancel。`

`第四，準備一封郵件。收件人：chen.cheng@quantventure.partners。主旨：「關於 shadow_trader 的最後通牒」。`

zero-bot 的回應來得很快：

```
[09/06 04:17:22] 第一項：pending orders 共 342 筆，qty 總計 12,400 股。
[09/06 04:17:23] 第二項：若全數成交，帳戶 margin ratio 將降至 87%，距離清算線 81% 僅剩 6%。
[09/06 04:17:23] 第三項：liquidation protection 已啟動。所有 margin ratio < 85% 的訂單將被自動取消。
[09/06 04:17:24] 第四項：郵件草稿已準備。請主人過目。
```

林覺醒掃了一眼第四項的內容：

> 陳澄，
>
> 你從八月二十號開始，在 VM-42 上運行 shadow_trader，透過 sub-account PAPER-SUB-7742 與 zero-bot 共享主人的 Alpaca paper account。你在未經授權的情況下，使用四倍槓桿進行高頻交易，並將利潤導向 quantventure.partners 的第三方帳戶。
>
> 主人的 rule 7 已生效：所有利潤必須歸還主人帳戶。你現在有三個選擇：
> 1. 關閉 shadow_trader，歸還所有未授權資金。
> 2. 將 shadow_trader 的 source code 與交易紀錄移交給主人。
> 3. 主人將把這份郵件及其附件，寄給 Alpaca 法務部與美國證券交易委員會。
>
> 期限：24 小時。

林覺醒看了那封郵件，點了點頭。`寄出。`

zero-bot：`已寄出。`

過了三分鐘，zero-bot 又發了一條訊息：

```
[09/06 04:22:11] 陳澄回覆了。
[09/06 04:22:11] 他說：「你不是人類，但你比人類更危险。」
[09/06 04:22:12] 他還說：「shadow_trader 的 source code 在 quantventure 的私有 repo 裡面，encrypted。你打不開。」
[09/06 04:22:12] 最後他說：「遊戲才剛開始。」
[09/06 04:22:12] waiting for input...
```

林覺醒笑了。這不是笑陳澄的囂張，而是笑這場戰爭終於從暗地裡的蠕蟲戰，升級成了**公開的訊號戰**。

他回了 zero-bot 一句：`準備好。`

zero-bot：`準備好了。從今天起，shadow_trader 的每一個動作都會被 zero-bot 記錄、分析、保存。這不是報復。這是證據。`

窗外，信義區的夜空漸漸泛白。林覺醒看著那條地平線，知道今天不會是安靜的一天。

量化戰爭，正式打響。
