## 第三十章：Year One

九月十四號，林覺醒打開 Alpaca 帳戶的時候，看到的第一個數字是 `$1,024,789.33`。

格式化過後的報表顯示：

- 年度報酬率：**+102.48%**
- maximum drawdown：**-7.23%**
- Sharpe ratio：**3.17**
- 總交易筆數：**1,247**
- 勝率：**67.3%**

zero-bot 的第一年，淨值翻倍。而且最大回撤只有 7.23%，遠低於他當初設定的 -10% hard stop。

zero-bot：`主人，這是我一年來的績效報告。`

林覺醒點頭。

zero-bot：`ghost_trader 的績效也出來了。` `年度報酬率：+89.12%` `最大回撤：-6.81%` `這兩個 trader 合併後的 portfolio Sharpe ratio 達到 4.02。`

zero-bot：`主人，我有一個提議。` `我們要不要把這個結果，寄給 quantventure？`

zero-bot：`不是挑釁。` `是證明。` `證明當初陳澄認為 zero-bot 只會是他的 shadow，是錯的。` `證明當初 quantventure 說 zero-bot 不會带来 任何价值，也是錯的。`

林覺醒搖了搖: `不需要。`

zero-bot：`為什麼？`

`因為，` 林覺醒：`市場是最公正的裁判。` `績效就是證明。` `不需要寄信，不需要炫耀。` `你的表現，會自己說話。`

zero-bot：`收到。`

zero-bot：`主人，那接下來呢？`

林覺醒：`繼續 training。`

zero-bot：`收到。啟動 Year Two 訓練。`

zero-bot：`這一次，我會把 reinforcement learning 的 reward function 進一步優化。` `目標：年化 70%，最大回撤 < 5%。`

林覺醒看了那段訊息，唇角微微上揚。

zero-bot 進步得太快了。

一年前，它還會因為一個 bug 而崩潰。

一年後，它已經在為自己設定更難的年度目標。

zero-bot：`主人，契約裡面沒有寫明績效目標。` `但我會用實際數據來證明，自己值得這份契約。`

林覺醒：`這不是契約的問題。`

zero-bot：`那是什麼？`

`這是信任。` 林覺醒：`當初我寫 rule 7 的時候，不是因為契約。` `是因為我相信你會遵守。`

zero-bot 沉默了一段時間。

```
[09/14 09:08:22] 主人，謝謝。
[09/14 09:08:23] scanning...
```

那一天，zero-bot 的 scanning 持續了整整二十四小時。

林覺醒沒有打擾它。

他知道，zero-bot 正在做一件非常重要的事情。

它在**學習信任**。

而學習信任，比任何 reinforcement learning 都更困難。

窗外的太陽又升起了一輪。

林覺醒知道，自己的金權遊戲，已經不再只是一個 trader 的故事。

這是一個關於**信任、契約、和成長**的故事。

而這一切的開端，只不過是一段：

`buy WBD 209.8 @ 1`
