# 橘子園 VIII： Dock C 的箱子

## Ch8 - 港口的暗影

隔天清晨，他們來到了 Harbour Warehouse #7。

这座仓库是港口最老旧的建築之一，外墙上的鐵鏽像是被時間燒過的痕迹。蕭拿著紙條，帶著林橙走進陰暗的走廊。

Dock C 的燈是壞的，只有遠處窗戶透進來的微光。

「這裡，」蕭在 C09 的前面停下來，手里握著那把從海底撈上來的鑰匙。

他把鑰匙插進鎖孔，轉動了一下。鎖打開的聲音在寂静的走廊裡回蕩。

箱子裡面是一個黑色的大信封。蕭把它拿出來，拆開。

裡面是一疊泛黄的相片、幾頁手写的筆記，還有一張用膠帶固定的、已經褪色的合照。

照片上的男人穿著深藍色的制服，身邊站著一個女人，懷裡抱著一個嬰兒。

「這是……」

「我父親、母親、還有我，」蕭的聲音很輕，「我從來没有看過這張照片。」

林橙看著他。他的手指輕撫照片上嬰兒的脸——那是他出生的樣子。

「另一邊是什麼？」她問。

蕭 flipped 過去。照片的背面寫著一行字：

> 保護他。保護她。保護真相。

「你父親愛你們，」林橙說。

蕭點點頭。他的眼睛裡有某种東西在閃爍。

「現在真相終於回來了。」

## Ch9 - 回来的信

筆記的內容讓林橙震驚了。

那是一封寫給萧的信，是母親在离开那天留下的：

> 親愛的蕭：
> 如果你看到這封信，代表你父親已經完成了他的使命。
> 鹽與鐵的 proof 已經在安全的地方。
> 現在交給你了。
> 請記住：永遠不要 trust 官方调查报告中的 version。那 version 是經過修改的。
> 真正的版本在我的心裡，也在那些照片裡面。
> 保護橘子園，保護愛你的人，保護 truth。
> 永遠的，媽媽。

信的下面是幾張港口官員與 overseas 商人見面的照片。照片的日期是2006年10月——正是他父親失蹤的前一個月。

「這些人……」林橙看著照片上的脸，「他們現在還在嗎？」

「有的还在職位上，」蕭的聲音變了，「有的已經退休，但 influence 還在。」

「你打算怎麼辦？」

蕭看著她。他的 eyes 里面有一種 finally 的堅定。

「把这些東西交給一個我可以信任的人，」他說，「然後我們回到橘子園，做我們該做的事。」

「橘子園？」

「是的，」蕭點點頭，「橘子園是 proof 的藏身之處。只要這片土地還在，真相就不會被埋沒。」

## Ch10 - 橘园的燈

兩個人回到橘子園的時候，已經是傍晚。

燈塔的光在遠處閃爍，這一次是連續的——平安的信号。

林橙站在橘子園的中央，看著那些在夕陽中閃爍的橘色果實。它們像是無數個小小的燈火，照亮了整個海岸。

「你看，」蕭的聲音從身後傳來，「橘子園的燈，從來沒有熄滅過。」

林橙回頭看著他。夕陽把他的輪廓染成金色，他的眼睛裡有 Finally 的 Peace。

「現在怎麼辦？」她問。

「我們把證據交給國際刑警，」蕭說，「然後我們继续守護這片橘子園。每一天，每一季，每一 year。」

「就這樣？」

「就這樣，」蕭微笑著，「這已經够了。」

遠處，海浪的声音漸漸安靜下來。燈塔的光在三點鐘方向穩定的閃爍，照亮了海岸的每一条 line。

三個弧線，一個圓點——信任的記号，終於被重新點亮。

而橘子園的燈，在黑暗中持續閃爍。

oranges only。永遠不變的信仰。
