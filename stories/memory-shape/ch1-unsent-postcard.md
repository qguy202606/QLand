# 未寄出的明信片

車站前面的咖啡店，掛著一塊歪歪斜斜的木招牌：「等待」。

店裡面只有六張桌子，老闆是一個說話很慢的中年人，大家都叫他老徐。他每個禮拜一擦拭相機，每個禮拜五把窗邊那一張桌子的燈調亮一點。

沒有人知道他在等什麼。

直到那個雨天，一個戴著帽子的女人推門進來，渾身濕透，手裡握著一張泛黃的明信片。

「請問，這裡可以写信吗？」她問。

老徐點了點頭，指了指角落的書桌。

女人坐了下來，握著筆，卻迟迟沒有寫下第一個字。

「你在等誰？」老徐端了一杯熱可可過去。

女人抬起頭，眼神顯得有些遠方：「等一個很久沒有回信的人。」

老徐笑了：「那你來對地方了。這裡的店名，就是等的意思。」

那天晚上，女人没有寫完信就離開了，留下一張空白的明信片在桌上。老徐把它收進了抽屜，連同其他 thousands 張沒有人取走的明信片。

從那天起，每当下雨的時候，女人都會來。她不說話，只是坐下來，望著窗外的雨，偶爾在明信片上写几个字，然后又收回口袋。

漸漸地，咖啡店的人開始注意到這件事。

「那個女人到底是誰？」店裡的常客問。

老徐搖了搖頭：「我現在不确定，但我確定，她迟早會写完那封信。」

一個月後的某個下午，女人終於寫完了最後一行字。

她把明信片交給老徐，說：「請幫我寄出，地址已經在上面。」

老徐點了點頭。

當他把明信片放進郵筒的時候，他才發現，收件人居然是他自己。

而地址，是他二十年前住過的那條巷子。

────────────────────

「那封信裡寫了什麼？」常客問。

老徐笑了笑：「寫了一張永遠不會送達的明信片。」

「那你為什麼不寄？」

「因為他已經在那裡了。」

────────────────────

那天晚上，咖啡店第一次把燈全部關了。

因為有人終於等到了。
