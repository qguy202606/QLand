## 第18章：顾客的記憶

記憶形狀咖啡店櫃台抽屜深處，藏著一個老旧的金属盒。

盒子是銅绿色的，边缘已经磨损，掛著一把小小的鎖。老板把它交給她的時候，說：「這裡面是每一位來過這裡的客人，留下來的一件東西。」

她打開盒子，發現裡面装滿了各種小物件：一支鋼筆、一张旧照片、一枚鈕扣、一段錄音帶、一張手寫的地址、一塊已經褪色的手帕。

每一件物品的下面，都壓著一張小卡片，寫著主人的名字和留下物品的原因。

她随手拿起一张照片。照片上是一对年輕夫婦，站在咖啡店的门口，笑容燦爛。

卡片上寫著：「這是我和妻子第一次約會的地方。她喝了一杯香草拿鐵，我喝了一杯黑咖啡。今天是她離開我的第三年。—— 阿明」

她望著那張照片，心裡有一種說不出的感動。

這就是記憶形狀真正的寶藏：不是咖啡，不是裝潢，而是這些被留下來的情感碎片。它們像星星一樣，照亮了這間小小的咖啡店，也照亮了每一個推開門進來的人。
