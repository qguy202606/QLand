# 晨光

ch5 之後，咖啡店的人漸漸多了起来。

不是因為什麼特別的原因，只是因為人们開始聽說，這裡有一家不會打烊的店，有一個人會幫你保管那些不敢寄出去的信。老徐沒有特别做什麼改變。他还是每天清晨五點半開門，打掃，磨豆，烤可頌。唯一的不同是，他的抽屜裡已經装满了明信片。責任感變重了，但他並未感到疲憊。反而有一種說不出來的踏實。

有時候他會在打烊後一張一張翻閱。每一張都是一個故事，每一個故事都是一個等待。沒有人催他，也沒有人要求他一定要回覆。但是他知道，只要有人把明信片放進來，就代表這家店還活著。代表還有信任存在於這間屋子裡。

第六個客人來的那個早上，沒有下雨，阳光透過窗戶灑進店里，照亮了每一張桌子。木質的地板反射出溫暖的光，空氣中混合著咖啡豆和新烤麵包的香氣。這種味道，老徐聞了二十年，永遠不會厭倦。

是一個中年男人，穿著簡單的襯衫，手裡拿著一個舊相框。他推門進來的时候，鈴鐺響得特別輕。他的步伐很穩，但眼神中藏著一絲疲憊。「請問……」男人的聲音很温柔，「這裡可以幫我保護一樣東西嗎？」

老徐點了點頭。

男人把相框放在柜台上。照片裡是一個年輕女人，站在這家店的門口，笑得很大声。那笑容很熟悉，老徐覺得好像在記憶深處見過。「這是我妻子，」男人說，「她走了五年了。這張照片是我們最後一張合影。」

老徐靜靜地聽著。

「我每個禮拜天都會過來，」男人說，「點一杯她最喜歡的拿鐵，坐一個小時。然後離開。」他的聲音很平静，但老徐看到了他眼底的淚光。「我不是來買東西的，」男人說，「我是來憶她的。」

老徐點了點頭。他知道這種感觉。因為他自己也是這樣等父親的。只是父親搬去了南部，他還偶爾能打電話。而這個男人的妻子，卻再也回不來了。

「你不是第一個這樣的人，」他說。

男人笑了笑：「我知道。我注意到這裡的抽屜越來越滿了。」

老徐沉默了一會兒，然後轉身從抽屜裡拿出一張明信片，遞給了男人。「這是？」男人問。

「是一張空白的明信片，」老徐說，「你可以寫給她。我會幫你保管。」

男人接過了明信片，點了點頭。

那天下午，男人寫完信之後，把明信片交了回來。他的字很工整，每一個筆畫都很穩重。老徐接過明信片，看了看上面的字：「親愛的，今天阳光很好。我還是會每個禮拜天來這裡，就像你以前一樣。等著下一次一起喝拿鐵。」

老徐把明信片放進了抽屜，在最上面。他知道，這家店已經不再只是一個賣咖啡的地方了。

它成了一個中轉站——承接那些來不及說出口的話，安放那些放不下的記憶。而每一個來這裡的人，都帶著自己的故事，和一張空白明信片。

老徐看著阳光洒在木地板上，心裡想著：「也許，等的盡頭，不是遺憾，而是理解。」

那一天，咖啡店的招牌第一次被阳光照亮。那兩個大字——等待——看起來，好像真的被等到了什麼。

門外的街道上，孩子們的笑聲遠遠傳來。風鈴再次響起。窗外，城市的喧囂依舊，但這裡依然安靜、溫暖，像一個永遠不關閉的港灣。

而那些曾经被遺忘的記憶，從今天起，都有了歸宿。
