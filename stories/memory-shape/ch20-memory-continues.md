## 第20章：記憶的延續

記憶形狀咖啡店開業的第十年，老闆決定退休。

他把店交給了她。

交接那天，老闆站在櫃台後面，望著她，微笑了。

「你知道我為什麼把店交給妳嗎？」老闆問道。

「因為我做得還不錯？」她開玩笑地說道。

「不是，」老闆搖搖頭，「是因為妳在這裡留下了自己的記憶。而這間店，從來就不屬於任何一個人。它屬於所有願意在這裡停下腳步的人。」

她望著那面牆、那些椅子、那台舊收音機、那個裝滿客人遺留物的金属盒子，忽然有了一種說不出的責任感。

她答應老闆，會繼續把這個地方經營下去。

老闆離開了，去了南方的一個小鎮。偶爾會寄來一張明信片，上面寫著：「咖啡店的貓還好嗎？—— 老夥計」

她把那張明信片貼在了留言簿的第一頁。

新的日子就這樣開始了。客人來來去去，故事起起伏伏，但記憶形狀的燈，依然在每一個下雨的夜晚，為路人亮著。
