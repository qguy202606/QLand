# 第五張桌子

ch4 的女生之後，咖啡店出現了一個新的規矩：

每到星期六下午，總會有一個老人出現在店門口。他穿著整齊的西装，手裡握著一把拐杖，走路的速度很慢，但每一步都很穩。老徐第一次見到他時，老人問：「這裡還有窗邊的位置嗎？」

老徐點了點頭。

老人坐了下來，點了一杯紅茶。他喝得很慢，一口一口，像是在品嚐什麼珍贵的東西。他的手指微微顫抖，杯子裡的茶水晃出小小的漣漪。

「我妻子以前最喜歡坐在這裡，」老人說，「她說窗邊的光線最適合看書。」

老徐沒有說話，只是靜靜地聽著。

「她走了三年了，」老人繼續說，「我每個禮拜六都會過來，幫她點一杯她最喜歡的紅茶。服務生換了好几個，唯獨老徐還在。老徐沒有阻止他，也從來沒有多問一句。」

老徐看了一眼抽屜。裡面已經有上百張明信片了，每一張都帶著一個名字，一個故事，一段無法寄出的思念。「你知道嗎？」老人彷彿看出了老徐的心思，「我覺得她並沒有真的離開。她只是換了一種方式存在。」

「什麼意思？」老徐問。

老人笑了笑：「你看這家店。每一個位子，每一盞燈，每一杯咖啡，都是她曾經存在的痕跡。只要這些東西還在，她就沒有真正離開。」

老徐沉默了。他想起了父親說過的話：「總有人會在等的。」原來，等的不是一個人，而是一段記憶。

那天晚上，老人離開時，老徐送了他一張明信片。「這是什麼？」老人問。

「是一張空白的明信片，」老徐說，「你可以寫給她。我會幫你保管。」

老人接過明信片，點了點頭。

老人走了之後，老徐坐在窗邊那张桌子上，靜靜地想著這家店的過去。他想起了父親，想起了第一批客人，想起了那些曾經來過、離開、又回來的人們。

這裡的每一張桌子，都見證過眼淚、微笑、告别和重逢。而他，只是那個願意一直坐在這裡，幫這些人保管記憶的人。

第二天早上，老徐在抽屜裡發現了一張寫好的明信片。是老人留下的。

上面寫著：「親愛的，這裡的燈光還是那麼溫暖。我每個禮拜六都會來，就像你以前一樣。期待下一次見面。」

老徐把那張明信片放了回去，夾在一本詩集的某一頁。他知道，這一次，店裡又多了一個願意等待的人。

而那張空白的明信片，終於有了屬於自己的故事。
