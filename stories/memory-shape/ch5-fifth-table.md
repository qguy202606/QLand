# 第五張桌子

ch4 的女生後來每個禮拜六都會來。

她不寫功課了，改為幫老徐整理明信片。那些放在抽屜裡、地址空白的明信片，被她一張一張拿出來，用彩色筆在旁邊畫小花、画小太陽、画小星星。

「你在做什麼？」老徐問。

「幫它們找地址，」女生說，「每張明信片都應該有一個地方可以去。」

老徐笑了笑，沒有說話。

有一天，女生翻到了一張特別舊的明信片。邊角已經發黃，上面的字跡也模糊了。但她還是勉強辨認出了幾個字：

「等你回來。」

沒有收件人，沒有地址，只有這四個字。

「這張最久了，」女生說，「應該先幫它找到地址。」

老徐接過那張明信片，看了很久。

「你知道嗎？」他說，「這家店原來有七張桌子。」

女生愣了一下：「可是我來的時候只有六張啊。」

老徐點了點頭：「第七張在裡面。」

他帶著女生走到廚房門口，推開那道門。裡面堆放著舊箱子、舊雜誌、一個壞掉的咖啡機，還有一張蓋著白布的桌子。

老徐把白布掀開。

那是一張木頭桌子，跟外面的六張一模一樣。只是桌面有一道很深很深的刮痕，像是有人曾經用力劃過什麼。

「這張桌子，」老徐說，「是我母親留下的。」

女生的眼睛亮了：「她也是這家店的客人嗎？」

「不，她是這家店的老闆。」

老徐說，父親生病之後，母親一個人撐了三年。那三年，她每天都坐在這張桌子後面，等著父親的病好起來。

但父親最終還是離開了。

母親沒有繼續開店。她把店頂給了老徐，自己搬到鄉下，種菜，養雞，過著安靜的日子。

「她說過，等待本身就是一種答案，」老徐說，「只是我們常常以為，答案必須是一個『結果』。」

女生似懂非懂地點了點頭。

那天晚上，她把那張寫著「等你回來」的明信片，小心翼翼地放在了第七張桌子上。

然後，她轉身對老徐說：

「老闆，我幫它找到地址了。」

「地址在哪裡？」老徐問。

女生笑了：「就在這家店。因為『回來』這兩個字，本身就是地址。」

老徐愣了一下，隨即笑了出來。

那是他這麼多年來，第一次笑得這麼大聲。
