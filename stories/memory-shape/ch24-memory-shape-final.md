## 第24章：記憶的形狀（第二部終）

十年後的一個雨天，她站在咖啡店門口，望著那盞熟悉的燈光。

花園裡的玫瑰花開了，風鈴在風中輕輕作響。屋內傳來客人們低低的談話聲，還有那台舊收音機傳出的爵士樂。

她想起剛開始的那一天——她推開門，老闆抬起頭， simply said：「隨便坐。」

那時候她什麼都沒有，只有一顆破碎的心和一個无处可去的身子。

如今，她有了這個地方、這些故事、這些願意停下腳步的人。

她抬起手，摸了摸門口掛著的風鈴。

叮咚。

那聲音像是一句問候，也像是一句告别。

記憶形狀的故事，不會在這裡結束。它會像那棵老榕樹一樣，繼續生长，繼續提供蔭涼，繼續等待下一個願意停下腳步的人。

而她，就是那個故事的守護者。
