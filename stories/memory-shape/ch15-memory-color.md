## 第15章：記憶的顏色

記憶形狀咖啡店有一本客人留言簿。

不是那种普通的留言簿，而是老闆亲手做的，封面是牛皮紙，裡面每一页都貼了一张小小的色卡。

老闆說：「每一個來這裡的人，都會带走一種顏色。那種顏色，就是你留在这里的記憶。」

她翻開留言簿，看到的第一张色卡是浅藍色的，下面寫著：

「第一次來，太紧张了，謝謝你讓我覺得這裡像家。—— 2024.03.12」

第二张是米色的：

「我在这里遇见了现在的妻子。那天下雨，我們共用一把傘。—— 2024.05.20」

第三张是淡紫色的：

「我丈夫走了之後，我每天都來。這家店的咖啡，讓我感覺他還在我身邊。—— 2024.09.03」

她望著這些色卡，忽然有了一種衝動——她也想留下自己的顏色。

她最後選定了橘紅色。像夕陽，像燈火，像她心目中那座城市最溫柔的模樣。

老闆幫她把色卡貼在留言簿的最後一页，然後微笑著說道：

「歡迎成為記憶形狀的一部分。」
