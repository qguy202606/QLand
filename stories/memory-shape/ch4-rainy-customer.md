# 雨中的常客

ch3 的男人之後，咖啡店又恢復了平靜。

但老徐發現，店裡的明信片越來越多了。有些是女人留下的，有些是老先生留下的，有些甚至是不認識的人趁老徐不注意時塞進抽屜的。

每一張明信片的背面，都沒有地址。

老徐沒有把它們郵寄出去。他知道，這些信不是用來寄的，而是用來存的。

第四個客人來的那個午後，沒有下雨，但空氣中彌漫著潮湿的氣息。

是一個穿著校服的女生，看起來像是高中生。她的頭髮濕濕的，手裡拿著一把折疊傘，走進來的時候，臉上的表情有些慌张。

「請問……」女生的聲音很小，「這裡可以借個地方寫作业吗？」

老徐點了點頭，指了指最裡面那張桌子。

女生坐了過去，把書包放在椅子上，從裡面拿出課本和筆記本。但写了五分鐘，她停了下來，轉頭看著老徐。

「老闆，」她問，「你這裡怎么這麼安靜？」

老徐笑了笑：「因為我喜歡安靜。」

女生點了點頭，又低下頭寫功课。可是不到一會兒，她又抬起了頭。

「老闆，」她再次開口，「你有沒有聽過『未寄出的明信片』這個故事？」

老徐的手頓了一下。

「誰跟你說的？」他問。

女生指了指抽屜的方向：「一個阿姨，她每個星期三都會來，坐在那裡，寫一張明信片，然後放進你的抽屜。」

老徐沉默了。

他從來沒有跟任何人說過這件事。

「那個阿姨說，」女生繼續說，「這裡的每一張明信片，都是一個未完成的約定。」

老徐靜靜地聽著。

「我來這裡，」女生說，「是因為我奶奶以前也是這家店的客人。她去年過世了，走之前跟我說，要是有一天我想她了，就來這家店坐坐。」

老徐突然明白了什麼。

「你奶奶是？」他問。

「她叫美惠，」女生說。

老徐點了點頭。他想起來了。那個總是帶著一本詩集，點一杯紅茶，卻從來沒有喝過的美惠。她每個禮拜六來，坐兩個小時，然後 leave，留下一張十元紙鈔在桌上。

「她說過，這裡的老闆不會趕人，」女生說，「而且這裡的燈光很溫暖。」

老徐的眼眶有些濕潤。

這一次，他沒有端飲料過去。他只是靜靜地站在櫃台後面，看著女生寫功课。

直到女生離開時，老徐才拿出一張明信片，寫下了一行字，放進了抽屜。

那是一句他父親說過的話：

「有些人不需要地址，因為他們早已經在你心裡。」
