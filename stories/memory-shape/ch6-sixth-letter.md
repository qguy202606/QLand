# 第六封信

女生叫林小雨。她每週六來，每週六離開，從不缺席。

老徐漸漸習慣了有她在的日子。以前店裡只有他一個人時，他總是把收音機調到最小聲，聽著新聞播報員機械的聲音，打發時間。但自從林小雨來了之後，他總會不自覺地把音量調大一點，因為她會問他：「這個新聞的後續呢？」

老徐不得不去查資料。

就這樣，咖啡店的櫃台旁邊多了一疊剪報。有火車誤點的後續報導，有新開咖啡店的評價，有一隻走失的貓被尋獲的新聞。

「老闆，你好像很喜歡幫助別人。」林小雨說。

「我只是想知道，」老徐回答，「那些人的等待最後有沒有結果。」

林小雨笑了：「老闆，你知道嗎？其實咖啡店就是一個收集結果的地方。」

老徐愣了一下。

「每一個來這裡的人，」林小雨說，「都是帶著某種期待來的。有人期待一杯好咖啡，有人期待一個安靜的地方，有人期待被理解。而你，就是那個讓他們期待被填滿的人。」

老徐沉默了許久。

「那我自己的期待呢？」他問。

林小雨沒有回答。她只是從抽屜裡拿出一張明信片，放在老徐面前。

明信片很舊，背面寫著一行字：「等你回來。」

「這張不是你的，」老徐說。

「嗯，是我奶奶留下的，」林小雨說，「她說，有些人等了一輩子，不是因為結果很重要，而是因為『等』這個動作本身，就是一種活着的方式。」

老徐拿起了那張明信片。

這一次，他沒有把它放回抽屜。他把它貼在了櫃台前面的牆上，就在那塊寫著「等待」的木招牌下面。

「老闆，」林小雨說，「你Change了。」

「什麼？」

「你以前只會把明信片放進抽屜，」林小雨說，「但今天，你把它掛出來了。」

老徐看著那張明信片，點了點頭。

「也許，」他說，「我也該讓自己的等待被看見了。」

那天傍晚，夕陽把咖啡店的玻璃染成了金色。

一個穿著灰色風衣的男人站在門口，手裡拿著一把雨傘。

是 ch3 的那個男人。

「我回來了，」他說。

老徐笑了：「我知道你會來。」

男人坐了下來，點了一杯熱可可。

這一次，他没有帶照片。他只是靜靜地坐著，望著窗外，偶爾點點頭，偶爾嘆口氣。

「我妻子以前說過，」他說，「這家店是她一生中，去過最美的地方。」

老徐的眼眶濕潤了。

他知道，這就是他一直等待的時刻。

不是為了某個特定的結果。

而是為了知道，所有的等待，最終都會以某種方式，回到起點。

雨開始下了。

但這一次，咖啡店的燈沒有熄。
