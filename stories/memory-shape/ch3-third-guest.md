# 第三個客人

ch2 的小女孩搬走之後，咖啡店安靜了兩個禮拜。

老徐還是每天清晨五點半開門，打掃，磨豆，烤可頌。但窗邊的上班族不再來了，角落的老先生只來了兩次，之後也消失了。

老徐沒有問。他知道，每一場離開都有自己的理由。

第三個客人來的那個晚上，下著傾盆大雨。

門被推開的時候，鈴鐺響得特別大。一個穿著灰色風衣的男人站在門口，手裡拿著一把濕透的雨傘，臉色蒼白。

「請問……」男人的聲音有些啞，「這裡可以借紙筆嗎？」

老徐點了點頭，指了指櫃台旁邊的書桌。

男人坐了下來，從懷裡拿出一張照片，放在桌上。照片上是一個女人，站在咖啡店門口，微笑著。

「這是我妻子，」男人說，「她最喜歡這家店。」

老徐靜靜地聽著。

「她去年過世了，」男人继续说，「她以前最喜歡坐在窗邊的位置，喝一杯熱可可。她說，這裡的燈光很溫暖。」

老徐看了一眼窗邊的座位，座椅的布套還是十年前那套，米白色，現在已經有點泛黃了。

「我上來台南出差，順便過來看看，」男人說，「她以前說過，如果有一天她不在了，要我代她來喝一杯。」

老徐轉身走進廚房，端出一杯熱可可，放在男人面前。

「這杯我請，」他說。

男人點了點頭，眼眶有些紅。

「謝謝。」

那天晚上，男人沒有說太多話。他只是靜靜地坐著，望著窗外，偶爾低頭看看那張照片。

雨停了的時候，男人站起身，把一杯錢留在桌上。

「我還會再來的，」他說。

老徐點了點頭。

他知道，這個店不只是賣咖啡的地方。它是收集眼淚的地方，也是安放記憶的地方。

每一張桌子，每一個座位，都有自己的故事。

而他，只是那個願意傾聽的人。
