# 咖啡店的名字

老徐的店沒有招牌，只有一塊木頭，上面用毛筆寫了兩個字：等待。

字是他父親寫的。父親當年也是開咖啡店的，在另外一條街，同樣的匾額，同樣的六張桌子。後來父親生病，把店頂給了老徐，自己搬到南部去住。

「店名不要改，」父親說，「總有人會在等的。」

老徐那時候年輕，不懂什麼叫「等」。他只知道每天清晨五點半開門，打掃，磨豆，烤可頌，等第一批客人進來。

日子一天一天過，他發現店裡開始出現一些固定的人。

一個是每周三來的老先生，坐最角落的位置，要一杯黑咖啡，看兩個小時的報紙，然後留下一枚硬幣在桌上，悄悄離開。

「那是誰？」剛來打工的服務生問。

老徐搖了搖頭：「不知道，但他從不拖欠。」

一個是每天早上一樣的上班族，坐窗邊的位置，點一杯拿鐵，把手提電腦放在桌上，卻從來沒有打開過。他只是坐十分鐘，喝一半，然後Leave。

「他在等什麼？」服務生問。

老徐搖了搖頭：「每個人都在等不同的東西。」

還有一個是下雨天一定會來的小女孩，轉學過來的，新學校還沒有朋友。她總是在放學後推開店門，點一杯熱巧克力，坐在角落畫畫。

老徐從不收她的錢。

「為什麼？」小女孩問。

老徐笑了笑：「因為你讓這家店變得更安靜。」

小女孩不懂，但還是天天來。

直到有一天，她再也沒有出現。

老徐等了很久，等到窗邊的拿鐵涼了，等到角落的黑咖啡變淡了，等到門口的雨停了。

他才知道，原來「等」這個字，有兩種寫法。

一種是等待別人來。

一種是等待別人走。
