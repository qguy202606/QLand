# 咖啡店的名字

老徐的店沒有招牌，只有一塊木頭，上面用毛筆寫了兩個字：等待。

字是他父親寫的。父親當年也是開咖啡店的，在另外一條街，同樣的匾額，同樣的六張桌子。後來父親生病，把店頂給了老徐，自己搬到南部去住。「店名不要改，」父親說，「總有人會在等的。」

老徐那時候年輕，不懂什麼叫「等」。他只知道每天清晨五點半開門，打掃，磨豆，烤可頌，等第一批客人進來。他心裡想著，只要店還開著，生意總會慢慢好起來。

日子一天一天過，他發現店裡開始出現一些固定的人。

一個是每周三來的老先生，坐最角落的位置，要一杯黑咖啡，看兩個小時的報紙，然後留下一枚硬幣在桌上，悄悄離開。「那是誰？」剛來打工的服務生問。老徐搖了搖頭：「不知道，但他從不拖欠。」

一個是每天早上一樣的上班族，坐窗邊的位置，點一杯拿鐵，把手提電腦放在桌上，卻從來沒有打開過。他只是坐十分鐘，喝一半，然後 leave。「他在等什麼？」服務生問。老徐搖了搖頭：「每個人都在等不同的東西。」

還有一個是下雨天一定會來的小女孩，轉學過來的，新學校還沒有朋友。她總是在放學後推開店門，點一杯熱巧克力，坐在角落畫畫。老徐從不收她的錢。「為什麼？」小女孩問。老徐笑了笑：「因為你讓這家店變得更安靜。」小女孩不懂，但還是天天來。

直到有一天，她再也沒有出現。

老徐等了兩個禮拜。窗邊的拿鐵涼了，角落的黑咖啡變淡了，門口的雨停了又下。他開始懷疑，是不是自己做錯了什麼，讓她不再來了。

第三個禮拜的星期三，老先生放了一張紙在桌上，上面寫著：「我女兒說她以前在這裡畫畫。她說老闆很好。謝謝。」

老徐把那張紙放進了抽屜。他沒想到，小女孩的故事，已經被別人記住了。

原來，等這個字的重量，從來不只在於時間的長短，而在於你看見了多少人的離開，卻仍然願意為他留一盞燈。

他才知道，原來「等」這個字，有兩種寫法。一種是等待別人來，一種是等待別人走。

但更多的時候，「等」是一種選擇。選擇相信，選擇留下，選擇讓這家店繼續亮著燈。就像父親說的那樣，總有人會在等的。
