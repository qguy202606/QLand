# 春日的回信

冬天快結束的時候，樂樂的手機收到一個訊息。是陳伯在醫院拍的，說膝蓋裝了新關節，開始復健了。

「你看，」樂樂把照片給阿明看，「陳伯說他終於可以站起來了。」

阿明舉起手機，對著照片笑了。

「那你打算怎麼祝賀他？」林姐問。

阿明想了一下，把畫本翻了出來。他在〈傳承〉那幅畫下面，補了一行小字：「願你再次走過夜市。」

然後他把畫寄給了陳伯。

兩個禮拜後，夜市出現了一個熟悉的身影。陳伯柱著拐杖，一步一步走向水果攤。年輕人跟在後面，手裡推著一台可折疊的輪椅。

「陳伯！」阿明迎上去。

陳伯笑了：「 ternary market 老了，但夜市還年輕。」

他指著攤子上陳列的芭樂：「今天的，由我來挑。」

阿明愣了一下，隨即點頭。

陳伯仔細地挑了三顆最大最甜的芭樂，交給年輕人。年輕人付了錢，但沒有等找零就離開了。

「記住，」陳伯說，「夜市從來不是一個人的。」

阿明把這句話寫在當天晚上的日記裡。

春天來了的時候，媽媽說阿公可以回家住了。林姐送來一盆小向日葵，放在店門口。

「希望你們每天都看到太陽。」林姐說。

樂樂也送來一張畫，畫的是阿明在攤子後面畫畫的樣子。題目寫著：〈畫畫的人〉。

阿明把這張畫和媽媽的一張舊照片放在一起。舊照片裡，巷口麵包店的老板正把一片吐司遞給小時候的媽媽。

跨世代的故事，在同一面牆上對望。

他知道，這個春天不只帶來溫暖，還帶來了新一季的夜市。

燈會亮。歌會唱。畫會繼續。

只要有人願意站在那個角落，一切就會延續。

就像那首歌一樣：「音樂只要還有人聽，就不會消失。」

而夜市只要還有人選那三顆芭樂，就不會結束。
