# 夜校

夏天結束的時候，女人問阿明：「你想不想繼續上課？」

阿明點點頭。於是他開始每個週末下午，去巷口七號學畫。

女人的名字叫林姐，以前是國小美術老師。生病之後，她辭了職，靠畫畫和賣一點手工藝品過日子。小女孩是她姪女，名叫樂樂，每個周末都會來。

「林姊為什麼不教別的學生？」阿明問樂樂。

樂樂搖搖頭：「她不喜歡太多人，她說真正想學的人，會自己找上門。」

阿明把那台修好的收音機帶到課堂上。林姐聽了那首老歌，靜靜地閉上眼睛。

「謝謝你讓它重新響起，」她說。

那天，林姐給了阿明一個新的主題：「畫一個你最想念的人。」

阿明畫了老伯。他沒有見過老伯年輕的時候，但憑著記憶裡老人坐在輪椅上微笑的樣子，他把線條勾勒出來。旁邊還畫了一顆芭樂，和一個小小的攤位。

林姐看了很久。

「你爺爺以前也是這樣賣水果的，」她說，「我把畫借回家，掛一個晚上，好嗎？」

阿明點點頭。

他知道，自己畫的不只是一位老人的臉，更是一種延續——夜市、芭樂、輪椅、歌曲，都還在。

那天晚上，他在畫紙背面寫了一句話：「想念從不因為沉默而消失。」

第二天，他把畫拿回來時，林姐在背面也寫了一句：「所以我們要一直說、一直畫、一直唱。」

阿明把那一頁夾回畫本裡。

從此以後，每個周末，他都會帶著新的畫過來，帶著新的故事。

有時候樂樂會畫他自己；有時候林姐會彈吉他（那是她最近剛學會的），讓音樂和畫一起流動。

這個城市還是有雨，但阿明不再只是等待雨停。他学会了自己走進巷口，推開那扇生锈的鐵門。

他知道，有人在等他。

而且他知道，無論多大的雨，都澆不滅一盞為愛亮著的燈。
