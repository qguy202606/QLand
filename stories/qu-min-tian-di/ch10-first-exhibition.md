# 第一次展覽

樂樂的學校舉辦年度美展，林姐答應幫樂樂準備作品。

「阿明要不要也一起參加？」林姐問。

阿明遲疑了一下。他從來沒有想過把自己的畫拿給別人看。

「就當練習，」林姐說，「畫你最真實的感覺。」

於是他畫了那個夏天的夜市：燈泡一盞一盞亮起來，媽媽的水果攤在角落，老伯的輪椅在照片裡，收音機放在旁邊，畫旁的題目是：〈傳承〉。

評審當天，林姐沒有陪他去。她說：「你自己去。」

展場在學校禮堂，四周都是小朋友的作品。阿明的畫看起來並不起眼，線條不夠圓潤，顏色也有點混。但每個經過的人，都會在夜市那幅前停一下。

有個老先生指著老伯的輪椅問：「這也是你自己寫的嗎？」

阿明點點頭。

「那時我也在夜市，」老先生說，「買過陳伯的芭樂。他總是多送我一顆。」

阿明低头看著自己的畫，忽然覺得，畫好像不只是畫。它是橋，把兩個從來沒有說過話的人，連在一起。

晚上，他把這個故事告訴林姐。

「這就是畫的意義，」林姐說，「讓看不見的，變看得見。」

第二天，林姐拿了一張給阿明。畫的人是她自己，坐在窗邊畫畫，陽光落在畫紙上。題目寫著：〈午後的开始〉。

她把畫送給阿明，說：「以後我們每年都一起參展，好嗎？」

阿明點點頭。

那張畫他帶回家，夾在賬本和舊收音機說明書之間。媽媽看到了，沒有說什麼，只是微笑。

從那天起，阿明的生活不再只有水果攤和畫本。他知道，無論將來走到哪裡，那盞為他亮的燈，一直在巷口。

他不必怕黑。

因為他已經學會，如何在雨裡，找到自己的方向。
